# CS Repository

A collection of data structures and algorithms.

## Build from source

### C++ project

To build C++ project
```bash
bazel build //x4444/...

# To build for  Debug and print build commands
bazel build -c dbg -s //x4444/...
```

### C project

To build C project
```bash
make [<target>]
```

### Java project

To build Java project
```bash
mvn clean install
```
